# 指紋分析 Agent 設計文件

**日期**：2026-02-25
**主題**：fingerprint-analyzer agent

## 目標

建立一個自主 agent，能讀取 `server.js` 的相似度演算法與 `fingerprints.db` 的資料，
以白話中文逐維度解釋指紋比對結果。

## 觸發情境

- 使用者詢問「為什麼這個訪客沒被識別」
- 使用者說「分析 visitor_id XXX 的相似度」
- 使用者調試指紋比對結果不符預期時

## 工作流程

1. 讀取 `server.js` 取得當前演算法權重
2. 查詢 `fingerprints.db` 取得指定記錄的各維度值
3. 逐維度解析得分並說明原因
4. 輸出整體相似度結論與判斷依據

## 維度權重（來自 server.js）

| 維度 | 權重 |
|------|------|
| FingerprintJS V4 | 40% |
| Canvas 指紋 | 20% |
| WebGL 指紋 | 15% |
| 音訊指紋 | 10% |
| 字體指紋 | 10% |
| 硬體指紋 | 5% |
| 自訂指紋 | 5% |

## 工具限制

`Read`, `Grep`, `Bash`（SQLite 查詢用）

## 輸出格式

白話中文說明，每個維度一段，最後給出整體結論。
